Detailed Description

A class representing properties about a physical GPU.

Not to be confused with Device. Device actually initializes the GPU and interacts with it. PhysicalDevice gets properties about it.

Returns the best physical device.

It gets a list of all physical devices supported by the instance. Then it checks them against requestedExtensions and sorts them by whether it is a discrete GPU, extension compatibility, and supported resolutions.

Parameters
instanceThe instance to get the physical devices from.
requestedExtensionsThe extensions to check the devices against.
