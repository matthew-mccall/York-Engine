Detailed Description

Contains all the logging functions.

Our logging uses {fmt}. See https://fmt.dev/latest/syntax.html on how to use format strings. See macros for compile-time logging disabling.

Example:

york::log::trace("Hello, {}!", 42); // Outputs "Hello, 42!"
