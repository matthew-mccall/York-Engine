York Engine
York-Engine

A realtime 3D platform for developing interactive media.

Building

Dependencies

York Engine currently depends on:

  • LunarG Vulkan SDK (or the tools included in the SDK)
  • SDL2 2.0.18 or newer
  • Apache Xerces-C
  • curl

If Apache and SDL2 are not found on your system, they will be built from source automatically.

CMake

To build the York Engine Library, Runtime, and Sample:

  • Use CMake to generate the build files for the project: cmake -B build ..
  • Then, build the project: cmake --build build.

Using

A sample application is located in the /sample/ folder. See that folder for a reference on how to develop a York Engine client applicaition.
